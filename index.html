Organização

- Começo esse tópico com uma pergunta: como Javascript é organizado da forma mais simples possível? Bom pensando nesssa pergunta façamos um paralelo a lingua portuguesa. por exemplo: como um livro é escrito em Português?

  • Em primeiro lugar temos a nossa sintaxe (forma de escrita)
  • As letras
  • Letras formam palavras
  • Palavras formam frases
  • Conjunto de frases formam um parágrafo
  • Alguns parágrafos formam uma sentença
  • Até chegar no conjunto final que é o livro (propriamente dito)

    • Já no Javascript temos uma situação bem parecida (como temos em todas as linguagens ou linguas), vou usar alguns console.log() pra explicar

  • temos em Javascript as sentenças de código que não necessariamente precisamos termina-las com ";" console.log("sentença de código").
  • temos os blocos de código, onde o código é escrito dentre as chaves{ console.log("bloco de código") }.

    • É basicamente isso, sentenças e blocos, porém existe uma infinidade de blocos e sentenças para se aprender

    Comentários

    - Toda linguagem de programação lhe permite comentar dentre as linhas de código ou nas próprias. o Javascript não é diferente, aqui o conteúdo é bem simples igual a forma de efetuar comentários em Javascript.

  • Temos comentários de apenas uma linha que são iniciados com // duas barras e a escrita vem logo a frente
  • Comentários de múltiplas linhas que começa com /* e fecha os comentários com */
  • E também podemos junto com os comentários de multiplas linhas incluir * no inicio de cada linha

    • Variáveis e Constantes

    Pra sabermos o que são variáveis em programação precisamos saber primeiro o que são dados e o que são informações. dado é uma informação inútil ou seja que ainda não tem nenhuma atribuição ou valor. por exemplo: o nome Bruno, se tivesse uma palavra Bruno em meio há outras informações aquela palavra seria um dado apenas.

    informação é o dado com atributo ou valor. por exemplo: Bruno é um homem de 28 anos. essa é a diferença entre dado e informação, a atribuição ou o valor que aquele dado tem em alguma semântica.

    A definição de variávelna programação se dá à um dado alocado na memória com um valor ou atribuição (nome)

  • Variáveis declaradas com a palavra reservada var tem escopo global ou de função.
  • Variáveis declaradas com a palavra reservada let tem escopo de bloco, instrução ou expressão.
  • As variáveis declaradas com a palavra reservada const devem ser iniciadas na declaração e não sofrem alterações.
